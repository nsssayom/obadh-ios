# README assets

Captured on a physical device running a **Release** build, in a real host app
(Notes) — never the in-app debug harness.

| File | Content |
|---|---|
| `notes-light.png` | Notes, light appearance, keyboard up, a line composed with Obadh |
| `notes-dark.png` | Same scene, dark appearance |
| `typing.gif` | Short screen recording of a line being typed (optional but wanted) |

Sample lines (Tagore / Nazrul):

- `amar sonar bangla, ami tomay bhalobasi` → আমার সোনার বাংলা, আমি তোমায় ভালোবাসি
- `bolo bir, bolo unnata mama shir` → বল বীর, বল উন্নত মম শির
